About The Project

Thank you for checking out our predictive algorithim! We are Luis, Aaron, Kate, and Pearce. Our project is a machine learning algorithm that predicts the prices of multiple cryptocurrency types such as etherium and bitcoin by using such concepts as tensor flow and has a massive database which is comprised of rows containing the prices of different cryptocurrency at different times which then uses these pieces of information to make a prediciton about what the next prices of these "coins" will be. Our project also has a second database which is structured as a relational model in order to store user profile information and other such bits of "user" information. Our database is mostly run by SQL and is queried by EJS and Jquery within our webpages and while it is not updated daily, we plan on implementing weekly updates to our information.
